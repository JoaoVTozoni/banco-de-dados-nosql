# banco-de-dados-nosql
Repositório para alocar o projeto da disciplina de Banco de Dados NoSQL. 

Catálogo Dinâmico de Produtos para E-commerce
Um sistema de retaguarda (backoffice) focado na flexibilidade de cadastro de produtos eletrônicos utilizando arquitetura NoSQL.

O Projeto
Este projeto consiste em uma aplicação simples para operações fundamentais de CRUD (Create, Read, Update, Delete) de um inventário de loja de tecnologia. O foco principal não está na complexidade das operações, mas na arquitetura da informação e na escolha do modelo de dados para resolver um problema clássico de e-commerce.
